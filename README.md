Dodgy game
----------

Simple dodgy game built with Rust and ggez game engine during a hackathon.

Usage
-----

[Install rust](https://rustup.rs). Run

    $ cargo run

If you are lazy, just let the bot to play it (activated with feature gate).

    $ cargo run --features=bot
